redis发布者订阅者模式

1. 什么是Redis

Redis是一款内存高速缓存数据库。Redis全称为：Remote Dictionary Server（远程数据服务），该软件使用C语言编写，Redis是一个key-value存储系统，它支持丰富的数据类型，如：string、list、set、zset(sorted set)、hash。

2. Redis特点

Redis以内存作为数据存储介质，所以读写数据的效率极高，远远超过数据库。以设置和获取一个256字节字符串为例，它的读取速度可高达110000次/s，写速度高达81000次/s。

3.Redis 发布订阅

Redis 发布订阅 (pub/sub) 是一种消息通信模式：发送者 (pub) 发送消息，订阅者 (sub) 接收消息。

Redis 客户端可以订阅任意数量的频道。

总结:

redis发布者订阅者缺点

  • 如果客户端订阅频道,但是速度不够快,不断积压会使redis缓冲区体积越来越大,使redis速度变慢,甚至直接崩溃.
  • 如果订阅方掉线,那么会丢失断线期间发布者发出的消息

应用

  • 多人在线聊天室
  • 公众号订阅等等
